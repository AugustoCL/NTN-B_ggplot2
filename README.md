## Visualização das Taxas de NTN-Bs c/ Cupons entre 08/2019 - 05/2020

Deixo aqui uma visualização das taxas de NTN-Bs com diferentes vencimentos negociados entre 08/2019 - 05/2020, utilizando dados do Tesouro Direto. Com ela é possível observar o impacto do atual cenário pandêmico no mercado de títulos brasileiros, onde destaco o início do stress ocasionado pelo COVID19 em março e, também, o stress político e institucional gerado pela saída do Sérgio Moro do Ministério da Justiça em abril. 

A visualização utilizou dados do Tesouro Direto extraídos pelo pacote GetTDData na linguagem R e o código produzido está disponível para reprodução e consulta no arquivo `'ntnbs.R'`
